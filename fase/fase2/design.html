Propósito

El propósito de este sitio es conocer el lenguaje para el diseños de sitios web y adquirir conocimiento HTML , en la de creación de páginas web, con el fin de afianzar cada uno de nuestros conocimiento el este lenguaje de programación. HTML5 es una tecnología que ha crecido con el fin de dar diferentes soluciones a nivel web y en codificar nuevas estructura para estas .
